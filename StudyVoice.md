## Project Details

**Project Title:** StudyVoice

**Project Description:** A zero-friction, voice-powered task and note manager built specifically for students. It utilizes the browser's native Web Speech API to allow users to capture ideas and log actionable tasks completely hands-free. Designed for mobile browsers, it features a split routing system for tasks vs. notes and uses offline-first local storage so no account or backend is required.

**Tech Stack Used:** React.js (Vite), Tailwind CSS v4, HTML5 Web Speech API, Browser localStorage
